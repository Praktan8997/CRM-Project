# Support CRM

Simple customer support CRM built using FastAPI + React + SQLite.

## Tech Stack

- FastAPI
- SQLite
- SQLAlchemy
- React
- Tailwind CSS

## Features

- Create tickets
- View tickets
- Search tickets
- Filter by status
- Update ticket status

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
